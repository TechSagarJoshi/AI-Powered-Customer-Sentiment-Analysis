# AI-Powered Customer Sentiment Analysis

An end-to-end NLP project that analyzes **568K+ Amazon customer reviews** using **Python, NLTK, Google Gemini API, and Power BI** to generate sentiment insights and support data-driven decision-making.

---

## Features

- Cleaned and preprocessed 568K+ customer reviews using Python and Pandas.
- Applied NLP techniques including tokenization, stopword removal, and lemmatization.
- Generated sentiment labels and performed exploratory data analysis (EDA).
- Integrated Google Gemini API to generate AI-powered customer insights.
- Built datasets ready for interactive Power BI dashboards.
